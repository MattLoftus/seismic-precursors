\section{Methods}
\label{sec:methods}

\subsection{Catalog and station data}
\label{sec:methods-catalog}

We use the International Seismological Centre (ISC) global bulletin
accessed via \texttt{obspy.clients.fdsn.Client} \citep{beyreuther2010}
for all six training regions and both test regions, with magnitude cutoff
$M \ge 2.5$ over 2000-01-01 to 2024-12-31. ISC merges contributions from
130+ regional network operators (JMA, IPOC, AFAD, INGV, SSN, AEC, NCSN/SCSN,
PNSN), giving region-specific completeness consistent with local
instrumentation density. Catalog ingestion at $M \ge 1.5$ specified in the
pre-registration is replaced here by $M \ge 2.5$ for computational
practicality, an explicit deviation flagged in the data-availability note.

The eight pre-registered region polygons span six tectonic regimes
(strike-slip, thrust, subduction, continental extension) and are taken
verbatim from \citep{loftus2026prereg}. Six are training regions; Mexico and
Alaska are held out as the test set.

\subsection{Magnitude of completeness per region}
\label{sec:methods-mc}

Per-region $M_c$ is estimated by maximum-curvature on the binned
frequency-magnitude distribution, plus the Woessner-Wiemer +0.20 correction
\citep{wiemer2000,woessner2005}, then iterated upward in 0.20-step
increments until the $b$-vs-$M_c$ curve plateaus, defined as
$|\Delta b| < 0.02$ between consecutive $M_c$ values. The plateau check is
itself a pre-registered diagnostic; the chosen $M_c$ is recorded as an
artifact and not retuned downstream.

\subsection{Declustering}
\label{sec:methods-decluster}

We apply the Zaliapin-Ben-Zion 2013 nearest-neighbor declustering algorithm
\citep{zaliapin2013} with the published default parameters
$b_{\text{ZBZ}}=1.0$, $d_f=1.6$, $p=q=0.5$, threshold
$\log_{10}\eta_0 = -5.0$. The auto-KDE-based threshold finder is computed
in parallel as a sensitivity check but does not enter the headline result.
Declustering is applied to both the full catalog (for catalog-derived
features) and to the target set itself (M $\ge$ 4.5 events kept only if
parent-less in the ZBZ proximity graph), per
\citep{loftus2026prereg} \S 1.4.

\subsection{Features}
\label{sec:methods-features}

Five catalog-derived features are computed per 30-day window:
\begin{enumerate}
  \item \textbf{$b$-value} (Aki MLE \citep{aki1965} at the per-region $M_c$,
        with the Shi-Bolt 1982 standard-error formulation \citep{shibolt1982}
        and a minimum sample size $N \ge 30$ events at or above $M_c$).
  \item \textbf{$b$-value drift}: the slope of $b$ across three equal-time
        sub-windows, requiring $\ge 10$ events $\ge M_c$ per sub-window.
  \item \textbf{Benioff strain accumulation}: $\Sigma \sqrt{E_J}$ over events
        in the window, with $\log_{10} E_J = 1.5 M + 4.8$ \citep{bowman1998}.
  \item \textbf{Benioff curvature}: the second-order coefficient of a
        quadratic fit to the cumulative $\Sigma \sqrt{E_J}(t)$ curve;
        positive corresponds to AMR \citep{bowman1998}.
  \item \textbf{Number of events $\ge M_c$} per window.
\end{enumerate}
Three waveform-derived features are also computed per window via a Round-C
sub-protocol: spectral slope $\alpha$ from a Welch PSD log-log fit
\citep{brune1970}, whitened-spectrogram Shannon entropy \citep{bensen2007},
and the median instantaneous frequency of the first IMF after empirical
mode decomposition \citep{huang1998}. For each window we sample six
10-minute snapshots evenly spaced across the 30-day span from the region's
primary broadband Z channel, preprocess each (instrument response removal
to ground velocity, bandpass 0.5--12 Hz clamped to $f_s/3$), compute the
three features per snapshot, and aggregate to per-window scalars (mean for
the spectrum and entropy, median for the IMF1 IF; robust to per-snapshot
fetch failures). Per-region FDSN routing reflects waveform-archive
realities: California's BK.PKD requires NCEDC, Italy's L'Aquila Observatory
post-2009 is MN.AQU on IRIS, and Cascadia/Turkey use IRIS directly. The
pre-registered repeating-event-rate feature
\citep[Nadeau-McEvilly]{loftus2026prereg} is excluded from the headline
panel.

\subsection{Null windows}
\label{sec:methods-null}

We define two null window types per region. Null A samples 200 random 30-day
windows that do not overlap any other target's $[t', t'+60]$ buffer
(aftershock-free) and are at least 30 days before any target's origin time;
the buffer is reduced from the v1 90-day zone to PRA-2's 60-day post-event
side only. Null B samples 200 random 30-day windows excluding only a 7-day
post-event window for any $M \ge 4.5$ event in the catalog. The colored-noise
synthetic null C is waveform-only and is not exercised in this paper.

\subsection{Evaluation protocol}
\label{sec:methods-eval}

For each (region, feature, null-type) we compute the receiver-operating-
characteristic area-under-curve (ROC-AUC) via the Mann-Whitney U statistic.
For each (feature, null-type) we then compute a macro-AUC across the
qualifying training regions (a region qualifies if it has $\ge 8$ kept
precursor windows after declustering and overlap rejection, per PRA-2
amendment 3).

Significance for the macro-AUC is assessed by three concurrent gates: (a)
the cross-region bootstrap CI95 lower bound exceeds 0.5, (b) the absolute
effect size $|AUC - 0.5|$ exceeds 0.05, and (c) the permutation $z$-score
(shuffling precursor/null labels independently within each region, 1000
permutations) exceeds 3 in magnitude. The pre-registered test-region step
applies the same protocol to Mexico and Alaska, then reports the test-region
macro-AUC as the headline result. The Bonferroni-corrected significance
threshold is $\alpha = 0.05 / (5 \text{ features} \times 2 \text{ nulls}
\times 2 \text{ test regions} \times n_\text{qualifying})$, adjusted post hoc
to the realized panel size per PRA-2 amendment 3.

\paragraph{Failure mode \#1 (overlap-rule density).} The pre-registered
overlap rejection rule was calibrated against California-scale event density
($\sim 5$ declustered M$\ge$4.5/year). At Cascadia (11/yr), Turkey
($\sim 6$/yr), and Italy ($\sim 2.5$/yr) the rule is workable, recovering
panels of size 36, 30, and 36 respectively after PRA-2 amendment 2 (drop
the pre-event side of the v1 forbidden zone). At Japan (78/yr) and Chile
(82/yr), 200+ overlapping forbidden zones cover the entire 9{,}131-day
calendar, leaving 1 window in each. The recovery from v1 to PRA-2 is
$3.2\times$ in pooled qualifying $N_\text{precursor}$, but Japan and
Chile are permanently dropped.

\paragraph{Failure mode \#2 (catalog $\times$ non-US $M_c$).} Under v1's ANSS
ComCat catalog, Japan $M_c = 4.55$ and Chile $M_c = 4.45$ — both at the
M$\ge$4.5 target threshold, rendering b-value features structurally
uncomputable in those regions. Switching to ISC under PRA-2 brings both
$M_c$ values down to 3.50, demonstrating that ISC's regional-network
merging fixes the teleseismic-only gap. The tradeoff: California $M_c$
rises from 2.75 to 3.50 because ISC has a coverage gap relative to
ComCat for US data. Net effect on California is positive (37 kept
precursor windows under PRA-2 vs 25 under v1).

\paragraph{Failure mode \#3 (b-feature $\times$ window-Mc incompatibility).}
At PRA-2's per-region $M_c$ values combined with 30-day windows, the median
$n$ of events $\ge M_c$ is 4-5 in California, Cascadia, and Italy and 76 in
Turkey. The $b$-value MLE's $N \ge 30$ floor is therefore satisfied in only
3\% of windows for the first three regions and 73\% in Turkey. The
\citet{gulia2019} b-drop precursor signal does not transfer to short windows
at moderate $M_c$ because $b$ is structurally uncomputable. The b-value
macro-AUC reported in Section~\ref{sec:results-macro} is therefore based
on degenerate per-region samples and does not constitute a clean test.

A noteworthy directional signal across the waveform half: \emph{all three}
waveform features sit at AUC $< 0.5$ versus Null A (precursor windows have
slightly lower spectral slope, entropy, and IMF1 IF than aftershock-free
random windows), with permutation $z$-scores of $-1.71$, $-1.56$, and
$-0.50$. The direction is consistent across all four qualifying regions for
spectral slope and entropy. None reach 3$\sigma$ individually, but the
common direction implies precursor windows are weakly more
source-radiation-like (more $f^{-2}$, more concentrated, lower-frequency
dominance) than aftershock-free random background --- consistent with a
sub-detection-threshold version of the seismic-driver hypothesis but well
below the pre-registered evidence threshold. We report this as exploratory.

The $b$-vs-Null A row appears suggestive but is the failure mode \#3
artifact identified above. A single-region exception worth flagging is
Turkey, where $b$ is reliably computable: $b$-vs-Null A $\text{AUC} = 0.37$
($z = -1.20$, sub-3$\sigma$), precursor $b$ lower than null A,
directionally consistent with \citet{gulia2019} but underpowered.

\subsection{Test-region results: failure mode \#4}
\label{sec:results-test}

Applying the same PRA-2 protocol to the held-out test regions: Mexico
(843 declustered M$\ge$4.5; 1 kept precursor window) and Alaska
(847 declustered; 1 kept). Both fail PRA-2 amendment 3's $N \ge 8$
minimum. \textbf{Zero of two test regions qualify}, and the cross-regional
test-region claim is therefore structurally inaccessible under the
pre-registered protocol.

This is failure mode \#4: the subduction-density structural issue
identified in failure mode \#1 extends to both pre-registered test regions.
The same regime-level incompatibility between high-rate subduction-zone
seismicity and the precursor-window framework that knocked Japan and Chile
out of training also knocks Mexico and Alaska out of test. The fact that
the failure mode replicates on a held-out test set is what gives it its
weight — it is not a training-curation artifact.

The strongest defensible cross-regional upper bound on precursor signal is
therefore the training-panel-only macro AUC of approximately
$0.51 \pm 0.05$ on the genuinely-computable features (Benioff total,
Benioff curvature, $n \ge M_c$). No test-region strengthening of this
bound is possible under PRA-2.
